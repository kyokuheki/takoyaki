# takoyaki
たこ焼き、卵焼きa.k.a明石焼きの作り方

※自分のためのメモ

## たこ焼き
500gの粉を買ってきた場合を想定し、粉を250g使う2～3人前

大体900~1000ccくらいのタネになるはず。粉250gで2000cc近いタネになり、50〜60個分くらい。

### 原料

- タネ ※()内は100gの場合
  - たこ焼き粉: 125g (100g)
  - 出汁: 563～625cc (450〜500cc)
    - 625で粉100:水500の比率、ちょっと焼きづらい
    - 563で粉100:水450の比率、慣れてなくても焼けるギリギリ
    - 濃いめにする。RIKEN和風だしの素だと、2スティック分くらい。
  - 卵: Mサイズ4個 (3個)
- 具
  - 青ネギ
  - 紅しょうが
  - 真だこ（茹で）
    - 生タコの場合は塩茹でする
  - 天かす/あげ玉
    - 丸い綺麗な形ではなく、惣菜揚げ物コーナーにあるような不揃いのやつが良い
- 薬味・ソース
  - 青のり
  - 鰹節
  - ソース
    - 少し甘いものがよい、オタフクのたこ焼きソースなど。なければオタフクのお好みソース

### 手順
#### タネ

この手順は、粉125g分を2回繰り返して、250gをタネにする。
いっぺんに粉250g作ろうとするとでっかいボウルが必要になって難しい。

- 冷めた濃いめの出汁625ccを作る
  - 冷ますのが面倒なので少量の熱湯で作って水を足す
  - ボウルに6スティック出汁の素を25cc分の約2倍分入れる。RIKEN和風だしの素だと、2スティック分くらい。
  - 75ccの熱湯で溶かす
  - 残り550ccの水を追加して冷ます
- 粉を混ぜる
  - 出汁に粉を混ぜる。粉に出汁を混ぜるとダマになって辛い。
- 卵4つを溶いてからボウルに入れる
- 5分位本気でかき混ぜる
- 冷蔵庫か野菜室で1~2時間程度寝かせる

備考

- 出汁は普通に作ってもいい
- 卵が少なくなると固まりにくくなるので水を減らす

#### 具の準備
- 青ネギ
  - 輪切り（小口切り）にする
- 紅生姜
  - 微塵切りにする
  - 紅生姜は水分をよく切っておくこと
- タコ
  - お好きな大きさで。1 x 2 x 2 くらいがちょうど良い気がする

#### 焼き
- 油を塗る
  - 専用のものがないならキッチンペーパーを丸めたものに油をつけてやると良い
- たこ焼き鉄板を強火で予熱する
  - 鉄板に少し生地を落としてジュッと音がするくらい
- タネを入れる
  - 入れる前にまぜる。底に粉が沈澱するため
  - 穴から溢れて鉄板が全く見えなくなるまで入れる
- 具材を入れる
  - タコを最初に入れる
    - 早いほどタコの味が生地に滲み出てうまい
  - 薬味系の具を入れる
    - ネギと紅生姜
    - 紅生姜は水分をよく切っておくこと
  - 天かすを入れる
    - ガサガサと豪快に入れる
    - 穴が天かすで見えなくなるくらい
  - この時点で鉄板から溢れそうになっているはず
- 強火で焼く
  - 穴の縁が盛り上がってくるまで待つ
- ひっくり返す
  - 穴の周囲に沿って切り込み入れる
  - 穴から溢れた部分を穴の中に押し込む
  - ひっくり返す、難しければ90度回転でも良い
  - 穴から溢れた部分はこの時点で穴に全部押し込む
- 強火で焼く
  - 最後まで強火！

## 卵焼きa.k.a明石焼き

なんちゃってレシピなので過信しないように。

### 原料

たこ焼きの倍くらいの卵で、粉が少なめになる。卵が多いので焼きやすい。

- タネ ※()内は100gの場合
  - たこ焼き粉: 125g (100g)
  - 水: 715cc (570cc)
  - 卵: Mサイズ7個 (6個)
  - 白出汁: 大さじ2.5 (2)
- 具
  - 真だこ（茹で）
    - 生タコの場合は塩茹でする
- 出汁 400cc
  - 水: 350cc
  - 白だし: 大さじ2
  - うす口しょうゆ: 小さじ1
  - 酒: 大さじ1
- 薬味
  - ミツバ

#### 焼き
- タネとタコの入れ方が違う
  - タネを溢れない程度に入れる
  - タコを入れる
  - タネを鉄板が見えなくなるまで入れる
- そのほかは概ね同じ。卵なのでたこ焼きのように丸くはならない
